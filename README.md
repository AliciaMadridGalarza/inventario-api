# 📦 API de Sistema de Gestión de Inventario Informático

¡Bienvenido al repositorio de **inventario-api**! Esta es una API REST diseñada para gestionar el inventario de equipos informáticos, licencias de software, componentes y asignaciones de usuarios en un entorno corporativo o institucional.

Desarrollado como un proyecto en **IntelliJ IDEA**.

---

## 🚀 Características Principales

*   **Gestión de Activos (Hardware):** Registro, actualización y baja de ordenadores, servidores, periféricos, etc.
*   **Control de Software:** Seguimiento de licencias disponibles, expiraciones y software instalado.
*   **Asignaciones:** Vinculación de equipos y componentes a empleados o departamentos específicos.
*   **Historial de Mantenimiento:** Registro de incidencias y reparaciones de los equipos.
*   **API RESTful:** Endpoints ordenados y listos para ser consumidos por cualquier frontend.

---

## 🛠️ Tecnologías Utilizadas

*   **IDE:** IntelliJ IDEA
*   **Lenguaje:** Java 
*   **Framework:** Spring Boot (Spring Web, Spring Data JPA, Spring Security) 
*   **Base de Datos:** MySQL / PostgreSQL 
*   **Gestor de Dependencias:** Maven

## 💻 Requisitos Previos

*   **Git**
